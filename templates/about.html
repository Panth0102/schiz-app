About This Project

This web application is a prototype developed for the early detection and monitoring of schizophrenia using artificial intelligence. It features a schizophrenia risk classification system powered by a logistic regression model trained on the publicly available COBRE dataset.

The goal of this project is to demonstrate how machine learning can be applied to assist clinicians in identifying potential early signs of schizophrenia through data-driven analysis.

Key features include:

  • Risk classification based on neuroimaging and symptom data
  • Patient-friendly symptom logging interface
  • Clinician dashboard with prediction outputs and reports

This is part of an academic research initiative aimed at integrating AI with digital mental health tools to improve accessibility and early intervention outcomes in psychiatric care.
